# คู่มือการติดตั้ง Student HRMS

## ขั้นตอนการติดตั้งอย่างละเอียด

### 1. ความต้องการของระบบ

ก่อนเริ่มติดตั้ง ตรวจสอบว่าคุณมีสิ่งเหล่านี้:
- **XAMPP** หรือ **WAMP** หรือ **MAMP** (สำหรับ Mac)
- **PHP 7.4** หรือสูงกว่า
- **MySQL 5.7** หรือสูงกว่า
- **Web Browser** (Chrome, Firefox, Edge)

### 2. ติดตั้ง XAMPP (สำหรับผู้เริ่มต้น)

1. ดาวน์โหลด XAMPP จาก: https://www.apachefriends.org/
2. ติดตั้งตามขั้นตอนปกติ
3. เปิด XAMPP Control Panel
4. กด **Start** ที่ Apache และ MySQL

### 3. แตกไฟล์โปรเจค

1. แตกไฟล์ `student-hrms.zip`
2. คัดลอกโฟลเดอร์ `student-hrms` ไปวางที่:
   - **XAMPP**: `C:\xampp\htdocs\`
   - **WAMP**: `C:\wamp64\www\`
   - **MAMP**: `/Applications/MAMP/htdocs/`

### 4. สร้างฐานข้อมูล

#### วิธีที่ 1: ใช้ phpMyAdmin (แนะนำสำหรับผู้เริ่มต้น)

1. เปิดเว็บเบราว์เซอร์ ไปที่: `http://localhost/phpmyadmin`
2. คลิกที่แท็บ **"New"** หรือ **"สร้างใหม่"** ด้านซ้ายมือ
3. ใส่ชื่อฐานข้อมูล: `student_hrms`
4. เลือก Collation: `utf8mb4_unicode_ci`
5. คลิก **"Create"** หรือ **"สร้าง"**
6. คลิกที่แท็บ **"Import"** หรือ **"นำเข้า"**
7. คลิก **"Choose File"** และเลือกไฟล์ `database.sql` จากโฟลเดอร์โปรเจค
8. เลื่อนลงมาด้านล่าง คลิก **"Go"** หรือ **"ดำเนินการ"**
9. รอจนกว่าจะขึ้นข้อความ **"Import has been successfully finished"**

#### วิธีที่ 2: ใช้ Command Line (สำหรับผู้ที่มีประสบการณ์)

```bash
# เปิด Command Prompt หรือ Terminal
cd C:\xampp\htdocs\student-hrms

# รัน MySQL command
mysql -u root -p < database.sql
```

### 5. ตั้งค่าการเชื่อมต่อฐานข้อมูล

1. เปิดไฟล์ `config/database.php` ด้วย Text Editor (Notepad++, VS Code, หรือ Sublime Text)
2. แก้ไขค่าต่อไปนี้ตามการตั้งค่าของคุณ:

```php
private $host = "localhost";        // ส่วนใหญ่ไม่ต้องแก้
private $db_name = "student_hrms";  // ชื่อฐานข้อมูลที่สร้าง
private $username = "root";         // username MySQL (ปกติคือ root)
private $password = "";             // password MySQL (XAMPP ปกติเป็นค่าว่าง)
```

**หมายเหตุ:**
- XAMPP: password เป็นค่าว่าง `""`
- WAMP: password เป็นค่าว่าง `""`
- MAMP: password อาจจะเป็น `"root"`

3. บันทึกไฟล์

### 6. ทดสอบการเข้าใช้งาน

1. เปิดเว็บเบราว์เซอร์
2. ไปที่: `http://localhost/student-hrms/`
3. หากทุกอย่างถูกต้อง คุณจะเห็นหน้า Login

### 7. เข้าสู่ระบบ

ใช้ข้อมูลเข้าสู่ระบบเริ่มต้น:
- **Username**: `admin`
- **Password**: `password`

## การใช้งานครั้งแรก

### 1. เปลี่ยนรหัสผ่าน Admin
1. เข้าสู่ระบบด้วย admin
2. ไปที่เมนู **โปรไฟล์**
3. เปลี่ยนรหัสผ่านเป็นรหัสผ่านที่คุณต้องการ

### 2. เพิ่มนักศึกษา
1. ไปที่เมนู **จัดการนักศึกษา**
2. คลิกปุ่ม **เพิ่มนักศึกษา**
3. กรอกข้อมูลให้ครบถ้วน
4. คลิก **บันทึก**

### 3. เพิ่มวิชาเรียน
1. ไปที่เมนู **จัดการวิชา**
2. คลิกปุ่ม **เพิ่มวิชาใหม่**
3. กรอกข้อมูลวิชา
4. คลิก **บันทึก**

### 4. ลงทะเบียนเรียน
1. ไปที่เมนู **ลงทะเบียน**
2. เลือกแท็บ **วิชาที่เปิดลงทะเบียน**
3. คลิกที่วิชาที่ต้องการ
4. เลือกนักศึกษา (ถ้าเป็น admin)
5. คลิก **ยืนยันการลงทะเบียน**

### 5. บันทึกคะแนน
1. ไปที่เมนู **ผลการเรียน**
2. คลิกปุ่ม **บันทึกคะแนน**
3. เลือกการลงทะเบียนที่ต้องการให้คะแนน
4. กรอกคะแนนกลางภาค, ปลายภาค, และงาน
5. คลิก **บันทึกคะแนน**

## การแก้ไขปัญหาที่พบบ่อย

### ปัญหาที่ 1: หน้าเว็บแสดง "Cannot connect to database"

**วิธีแก้:**
1. ตรวจสอบว่า MySQL ใน XAMPP เปิดอยู่หรือไม่
2. ตรวจสอบค่าใน `config/database.php` ว่าถูกต้อง
3. ลองรีสตาร์ท Apache และ MySQL

### ปัญหาที่ 2: หน้าเว็บแสดงเป็น Error 404

**วิธีแก้:**
1. ตรวจสอบว่าโฟลเดอร์ `student-hrms` อยู่ในตำแหน่งที่ถูกต้อง (htdocs)
2. ตรวจสอบว่า Apache ใน XAMPP เปิดอยู่
3. ลอง URL: `http://localhost/student-hrms/index.php`

### ปัญหาที่ 3: ไม่สามารถ Login ได้

**วิธีแก้:**
1. ตรวจสอบว่าฐานข้อมูลถูกสร้างและ import ข้อมูลแล้ว
2. ลองใช้ phpMyAdmin ตรวจสอบว่ามีข้อมูลในตาราง `users`
3. หากไม่มีข้อมูล ให้ import ไฟล์ `database.sql` อีกครั้ง

### ปัญหาที่ 4: Session หายหรือไม่ทำงาน

**วิธีแก้:**
1. ตรวจสอบว่าโฟลเดอร์ `C:\xampp\tmp` มีอยู่และมีสิทธิ์เขียน
2. ลองล้าง Cache และ Cookie ของเบราว์เซอร์
3. ปิดเบราว์เซอร์แล้วเปิดใหม่

### ปัญหาที่ 5: ภาษาไทยแสดงเป็นอักขระพิเศษ

**วิธีแก้:**
1. ตรวจสอบว่าฐานข้อมูลใช้ charset `utf8mb4`
2. เปิดไฟล์ PHP ด้วย editor ที่รองรับ UTF-8
3. เพิ่ม `<meta charset="UTF-8">` ในทุกหน้า HTML (มีอยู่แล้วในโปรเจคนี้)

## การปรับแต่งเพิ่มเติม

### เปลี่ยนชื่อระบบ
แก้ไขไฟล์ HTML ทุกไฟล์ โดยค้นหาคำว่า "Student HRMS" และเปลี่ยนเป็นชื่อที่ต้องการ

### เปลี่ยนสี Theme
แก้ไขไฟล์ CSS หรือแก้โค้ดใน `<style>` ของแต่ละหน้า:
```css
/* หาและเปลี่ยนโค้ดนี้ */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
/* เป็นสีที่ต้องการ */
```

### เพิ่ม Logo
1. วาง logo ไฟล์รูปในโฟลเดอร์ `assets/images/`
2. แก้โค้ด HTML แทนที่ไอคอน `<i class="bi bi-mortarboard-fill"></i>` ด้วย:
```html
<img src="assets/images/logo.png" alt="Logo" style="height: 40px;">
```

## เพิ่มเติม: การ Deploy บน Server จริง

หากต้องการนำระบบขึ้น Server จริง:

1. **อัปโหลดไฟล์**
   - อัปโหลดทุกไฟล์ไปยัง web hosting
   - ใช้ FTP Client เช่น FileZilla

2. **สร้างฐานข้อมูล**
   - ใช้ cPanel หรือ phpMyAdmin บน hosting
   - Import ไฟล์ `database.sql`

3. **แก้ไขการเชื่อมต่อ**
   - แก้ไฟล์ `config/database.php`
   - ใส่ข้อมูลที่ hosting ให้มา

4. **ตั้งค่าความปลอดภัย**
   - เปลี่ยนรหัสผ่าน admin
   - ตรวจสอบสิทธิ์ไฟล์และโฟลเดอร์
   - เพิ่ม SSL Certificate (HTTPS)
